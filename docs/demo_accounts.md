## 使用指南

### 管理员（admin）操作演示流程
1. 访问 http://localhost:5000，使用 admin/123456 登录
2. 工作台页面查看系统风险告警横幅
3. 进入「用户管理」审核待审核用户，修改角色/密码
4. 进入「审计日志」查看业务流水和系统操作日志

### 教务（staff）操作演示流程
1. 使用 staff/123456 登录
2. 进入「学员管理」新增/修改学员档案
3. 切换到「报名与缴费」tab，为学员报名
4. 录入缴费记录（选择定金/尾款/退款）
5. 进入「课程与班级」查看/管理课程和班级信息

### 教师（teacher）操作演示流程
1. 使用 teacher/123456 登录
2. 进入「作品管理」查看学员提交的作品
3. 点击「点评」按钮，为作品打分并添加评语

### 学员（student）操作演示流程
1. 使用 student/123456 登录
2. 工作台查看作品截止提醒
3. 进入「学员管理」查看个人档案和报名记录
4. 进入「作品管理」查看提交的作品和点评结果
5. 点击「提交作品」按钮，新增作品提交
